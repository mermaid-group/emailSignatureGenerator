Introduction & Instruction
This web page has been provided for Mermaid staff to create standardized email footer.
Please refer to the How-To video / document, and proceed to generate and apply your personalized email footer in Outlook program.

Note on Internet Browser Program
Chrome is the recommended browser.
For Internet Explorer, you may need to manually click the 'Allow blocked contents' button at the bottom of the program screen for the generator to work properly.
Browsers other than Chrome and Internet Explorer, e.g. Safari, Firefox, Opera, etc. are NOT supported by this self-service page.

Note on Email Client Program
The footer created from this page is intended for use with Outlook program only. The pasting of contents with other email client has never been tested/verified.
